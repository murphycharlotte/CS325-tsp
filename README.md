# CS325-tsp
to compile code
enter: make tsp

to run example 1
enter: make example1

to run example 2
enter: make example2

to run example 3
enter: make example3

to run all 3 examples
enter: make all_examples

to run test *  , where * is 1 through 7 inclusive
enter: make test*

to run all tests
enter: make all_tests

to remove all object files, .tour files, and tsp object file
enter: make clean
